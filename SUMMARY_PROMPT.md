# Market Pulse Summary Prompt

Скопируй этот промпт + дайджест и отправь в Claude.

---

## Промпт

```
Ты — криптоаналитик. Проанализируй этот дайджест крипто-новостей и сделай:

1. **Краткую выжимку** (5-7 ключевых событий, 1-2 предложения каждое)

2. **Настроение рынка** (bullish/bearish/neutral + почему)

3. **На что обратить внимание** (1-3 пункта что может повлиять на рынок)

Формат: кратко, по делу, на русском. Без воды.

---

ДАЙДЖЕСТ:

[вставь сюда дайджест]
```

---

## Пример использования

1. Получаешь дайджест в Telegram
2. Копируешь текст дайджеста
3. Открываешь Claude (claude.ai или CLI)
4. Вставляешь промпт выше + дайджест
5. Получаешь структурированный саммари

---

## Альтернативные промпты

### Быстрый (только выжимка)
```
Сделай краткую выжимку этих крипто-новостей на русском (5-7 пунктов):

[дайджест]
```

### Для трейдера
```
Проанализируй новости с точки зрения трейдера:
- Какие активы могут вырасти/упасть?
- Есть ли сигналы для входа/выхода?
- Ключевые риски?

[дайджест]
```

### Для инвестора
```
Проанализируй новости с точки зрения долгосрочного инвестора:
- Фундаментальные изменения в индустрии
- Регуляторные тренды
- Институциональная активность

[дайджест]
```
